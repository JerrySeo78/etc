01

Claude는 기본적으로 대화가 끊기면 이전 내용을 기억하지 못합니다. 새 세션을 열 때마다 매번 이런 경험을 하게 되죠.

03

준비물 체크

필요한 건 딱 3가지뿐입니다. 모두 무료이고, 5분이면 준비 끝!

준비물 구분 설명
Desktop Commander 필수 Claude가 PC 파일을 읽고 쓸 수 있게 해주는 MCP 서버입니다. Claude 데스크톱 앱의 설정 → 통합(Integrations)에서 설치합니다.
Claude 프로젝트 필수 claude.ai에서 프로젝트를 하나 만들고, "Custom Instructions"에 규칙을 넣습니다. Pro 플랜 이상 필요.
PC 폴더 하나 필수 아무 폴더나 OK. 예: C:\Claude 또는 D:\Projects\Claude
Google Drive 등 선택 업무 문서를 Claude가 참조하게 하고 싶으면 연결. 없어도 핵심 기능은 동작합니다.
포인트: Obsidian이나 Notion 같은 별도 앱은 필요 없습니다. 그냥 PC에 폴더 하나, 파일 하나면 충분해요.
04

세팅 방법 (5분)

아래 3단계만 따라하면 세팅 완료입니다.

1
Desktop Commander MCP 설치
Claude 데스크톱 앱 → 설정(Settings) → 통합(Integrations) → Desktop Commander 추가.
이걸 켜면 Claude가 내 PC의 파일을 읽고 쓸 수 있게 됩니다.
2
DASHBOARD.md 파일 만들기
원하는 폴더에 DASHBOARD.md 파일을 하나 만들어주세요. 내용은 비어 있어도 괜찮고, 아래 예시를 복사해서 넣어도 됩니다. Claude에게 "대시보드 초기화해줘"라고 하면 알아서 채워줍니다.
DASHBOARD.md 예시 # 대화 현황판 ## 프로젝트 현황 | 프로젝트 | 상태 | 최근 작업 | 다음 할 일 | |---------|------|----------|-----------| ## 최근 대화 요약 (Claude가 자동으로 채웁니다) ## 진행 중인 작업 - [ ] ## 세션 로그 (세션마다 자동 추가됩니다)
3
프로젝트 Custom Instructions에 규칙 넣기
claude.ai → 프로젝트 → 설정 → Custom Instructions에 아래 내용을 붙여넣기하면 끝!
Custom Instructions 내용 ## 세션 시작 프로토콜 사용자가 작업을 시작하면: 1. Desktop Commander로 대시보드 읽기 경로: C:\Claude\DASHBOARD.md (본인 경로로 변경) 2. "최근 대화 요약"을 확인 3. 이전 맥락을 자연스럽게 이어서 대화 ## 세션 종료 프로토콜 "정리해줘", "저장해줘" 요청 시: 1. 이번 대화 내용 요약 2. DASHBOARD.md 업데이트 - 프로젝트 현황 갱신 - 세션 요약 추가 - 다음 할 일 정리 3. 사용자에게 간략히 보고

06

자주 묻는 질문

Obsidian이 꼭 필요한가요?
아닙니다. Obsidian은 전혀 필요 없습니다. Desktop Commander는 PC의 아무 폴더에 있는 파일이든 읽고 쓸 수 있기 때문에, 메모장으로 만든 .md 파일이면 충분합니다.
Google Drive에 대시보드를 두면 안 되나요?
현재 Google Drive MCP는 "읽기 전용"이라 Claude가 자동으로 업데이트를 못 합니다. 그래서 PC 로컬 파일이 가장 좋습니다. Google Drive는 업무 참조 문서를 읽는 용도로 쓰면 좋아요.
Mac에서도 되나요?
네! Desktop Commander는 Windows, Mac, Linux 모두 지원합니다. 경로만 OS에 맞게 바꿔주면 됩니다. (예: Mac은 ~/Documents/Claude/DASHBOARD.md)
프로젝트가 여러 개면 어떻게 하나요?
같은 폴더에 프로젝트별 .md 파일을 추가로 만들면 됩니다. 예를 들어 DASHBOARD.md는 전체 현황판으로 두고, project-a.md, project-b.md 같은 상세 파일을 따로 관리할 수 있어요.
무료 플랜에서도 되나요?
Desktop Commander MCP와 프로젝트(Custom Instructions)는 Claude Pro 플랜 이상에서 사용 가능합니다. 무료 플랜에서는 Claude 메모리만 활용하는 제한적인 방법을 쓸 수 있어요.
내 파일이 Claude 서버에 저장되나요?
아닙니다. DASHBOARD.md 파일은 오직 내 PC에만 저장됩니다. Claude는 대화 중에 Desktop Commander를 통해 읽고 쓸 뿐, 파일 자체를 서버에 업로드하지 않습니다.
